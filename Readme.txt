Instagram clone - demo: http://instaclone.sentebiolab.com

Simple instagram clone written by ASP.NET MVC 5.
To run on localhost please use visual studio 2017 - open InstaClone.sln and run using Run/Debug.
VS will automatically download required packages

To publish to IIS server - please edit web.config: database path, ssl and other configurations
Then either publish to cloud or local disk and move the folder to your iis domain folder

Some ideas were taken from https://github.com/Package/Insta-Sharp, but the app
is simple enough not to use services and repositories.

Клон - инстаграма

Простой клон инстаграма написан на ASP.NET MVC 5
Чтобы запустить на localhost нужен visual studio 2017 - открыть InstaClone.sln и запустить через Run/Debug.
VS автоматически скачает недостаюшие библиотеки. 


Чтобы запустить в продакшн - нужно изменить web.config, изменить базу данных итд,
потом сделать Publish и загрузить на IIS сервер.

Некоторые идеи были взяты с https://github.com/Package/Insta-Sharp
